# adec
a simple bash tool to decompile Android apps faster
 
Copyright (c) 2017 - Giulio Mazzanti <mazzantibox@gmail.com>

Based on initial idea by Marco Ronzano

Projects wrapped:
- dex2jar
- jd-gui

Java and adb must be installed in your $PATH.
